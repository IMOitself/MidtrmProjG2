Concept

Study Hub is a community-driven site where students in the Web App Course can share study tips, post their weekly schedules, and collect useful tools and resources. The goal is to help learners discover the right tools for coding, design, and assets so they can build better projects and study more efficiently.

Tools (organized by category)

Below is a compilation of tools categorized by their primary use — IDEs, design, and assets.

Online IDEs

  • Replit — collaborative online IDE for many languages.
  • CodePen — front-end playground for HTML, CSS, and JS.
  • JS Fiddle — quick JavaScript/HTML/CSS prototyping.

Design

  • Draw.io — diagrams and flowcharts.
  • CSS Grid Generator — visually create CSS grid layouts.
  • Palette Generator (Coolors) — quickly generate color palettes.
  • Google Fonts — free web fonts library.
  • Wireframing (Whimsical) — fast wireframing and flow diagrams.

Icons & Assets

  • HD Images (Unsplash) — free high-resolution photos.
  • Icons (Flaticon) — large collection of icons and vectors.

Categories For Tools (quick overview)

  • Design: Draw.io, CSS Grid Generator, Palette Generator, Google Fonts, Wireframing
  • Icons & Assets: Unsplash, Flaticon

Example Weekly Study Schedule (suggested)

  1. Monday — Review lecture notes; practice HTML/CSS basics (1.5 hrs)
  2. Tuesday — Work on lab exercises; try an online IDE (2 hrs)
  3. Wednesday — Study JavaScript concepts; small exercises (1.5 hrs)
  4. Thursday — Design session: wireframing & palettes (1 hr)
  5. Friday — Group study / code review (2 hrs)
  6. Saturday — Project work / asset gathering (3 hrs)
  7. Sunday — Rest, light reading, plan next week (30–60 mins)
